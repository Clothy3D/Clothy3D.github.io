Image Processing

PNG Channel Mixer​

This tool is used to create a png image whose each change comes from a channel of an input png image or is set to a constant using the following parameters:

  • Output: output file
  • Channels: number of channels of the output image
  • Input Channel #:
    • Input image
    • Channel of input image
    • Const: if checked, set to a constant value
    • Value: constant value

After setting the parameters, clicking the "Process" button will do the processing.

PNG Resier​

This tool is used to resize a png image using following parameters:

  • Input: input file
  • Resampling Factor
  • Output: output file

After setting the parameters, clicking the "Process" button will do the processing.
