Thresholding And Irrelevant

Clicking these button sets the threshold and related face attributes used by face matching.

Set Threshold​

This button allows setting the threshold using a dialog window:

= Threshold​

Clicking this button sets the heights of selected faces, or all visible faces if no faces are selected, to be equal to the threshold.

> Threshold​

Clicking this button sets the heights of selected faces, or all visible faces if no faces are selected, to be higher than the threshold.

< Threshold​

Clicking this button sets the heights of selected faces, or all visible faces if no faces are selected, to be lower than the threshold.

Irrelevant​

Clicking this button sets the selected faces, or all visible faces if no faces are selected, to be irrelevant, i.e., not participating in the face matching.
