GUI Options

Ask For Modifying Shape Control Point Distances​

This is an option used by Dress3D when loading a cloth asset.

Capture Screen As Saved File Icon​

This is used when export a model to a file. If checked, an image file and its 128x128 icon file are created by capturing the viewport.

Give Warning Before Delete Meshes​

If checked, a warning is given before a mesh is deleted.

Show Measurements​

This is used when entering vertices. If checked, the the distances (line, X, Y, Z) between the selected vertices are displayed as shown in the following figure:

Point Radius (mm)​

This is the radius of the red ball indicating the selection of a vertex or displaying skin weights.

Radius For Adjusting Vertices (mm)​

This is the configuration parameter used by adjusting vertex positions. It is the radius of the ball containing the adjusted vertices, and centered at the clicked point.

Depth For Adjusting Vertices (mm)​

This is the configuration parameter used by adjusting vertex positions. It is the depth below the clicked point.

Height Scale For Adjusting Vertices​

This is the configuration parameter used by adjusting vertex positions. It is the maximal adjusting range.

Skeleton Bone Axes Length​

This is the axis length of the bone when showing/editing skeleton.

Auto Adjust Camera Ranges and Camera Near Clip Range (%)​

If checked, the camera near clip range is automatically adjusted to ensure the models are completed visible.

If not checked, the camera near clip range can be adjusted using Camera Near Clip Range (%) slider. This is often used to facilitate the viewing or editing faces/edges on the inner surface of a model.
