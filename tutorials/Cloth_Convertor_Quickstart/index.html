Cloth Convertor Quickstart

This tutorial introduces the user interface and basic operations of the Cloth Convertor tool.

It shows how to convert a cloth model to an interchangeable cloth asset.

Input:

  • Models:
    • original cloth exported by CLO/MD (Marvelous Designer)
  • Assets
    • corresponding deformer template

Output:

  • Assets
    • converted interchangeable cloth

This guide shows how to convert a cloth exported by CLO/MD (Marvelous Designer) using the Cloth Convertor tool through the following process.

The process shown can also be applied to convert clothes exported by other softwares such as Daz3D, iClone and Unreal Engine (UE).

0. Export A Cloth​

Firstly export a cloth from a CLO project to an obj file using CLO's export function.

1. Import The Cloth​

Then import it to the middle scene using the default import parameters.

2. Wrap An Interchangeable Base Character To Body Inside Cloth​

Find and download the "MD_Female_1" deformer template from the marketplace.

Import and apply the template. The character in the right scene will automatically deform to wrap the character in the middle scene.

3. Delete Body Inside Cloth​

Delete the body mesh inside the cloth using operations 1 and/or 2.

4. Check and Merge Cloth To An Isolated Mesh​

THis cloth consist of only one isolated mesh, so do nt need to merge.

5. Configure and Process​

Start the processing flow, in each step:

Step 1. Select Supporting Body Type​

Select the "Inside Tight" body type.

Optionally export.

Step 2. Set Parts and Parameters​

If you have not changed the default setting, just use it. Otherwise click the "Reset Parameters" button to reset.

Optionally export.

Step 3. Calculate/Modify Shape Control Points​

Click the "Calculate Shape Control Points" button to calculate, then check by showing them which shows some points are inside the body.

Click the "Modify Shape Control Points" button to modify, then check by showing them which shows all points are outside the body.

Then optionally click the "Export Shape Control Points" button and the "Export Modified Shape Control Points" button to export to files.

Step 4. Calculate Skin Transfer Points​

Click the "Calculate Skin Transfer Points" button to calculate, then check by showing them.

Then optionally click the "Export Skin Transfer Points" button to export to a file.

Step 5. Export​

Export results to disk.

Step 6. Verify by Testing​

Dress the interchangeable cloth on interchangeable base characters using Dress3D, and ensure the results are normal.
