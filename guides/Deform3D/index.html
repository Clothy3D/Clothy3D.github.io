Deform3D Guide

Deform3D is a topology transfer tool to wrap any 3D static model to another, and transfer textures. It is used in the following steps.

1. Import A 3D Static Model​

Import the target model to the middle scene.

2. Import Another 3D Static Model​

Import the source model to the right scene.

2. Edit The Models (Optional)​

This step is optional.

You can edit the two models using any of the following operations in any order:

  • hide or delete unused meshes using operations 1 2 3 4 5 6

    For example, you may hide any clothes and weapons.

  • duplicate and copy, align, flip Smooth and decimate, and decompose the meshes

  • translate/rotate/scale meshes using gizmo

  • hide unused faces using operations 1 2 3

  • adjust vertex positions

4. Merge Meshes​

Merge the meshes in the right scene.

5. Stitch Faces (optional)​

This step is optional.

To keep the geometric structure of the mesh unchanged during deforming, you may sometimes need to stitch faces of the mesh in the right scene using 1, 2 and 3.

5. Wrap Source Mesh To Target Model​

Wrap the mesh in the right scene to the model in the middle scene by one of the following approaches:

Deform Using A Deformer Template​

This approach is recommended to try first as it is faster.

If you have a suitable deformer template which is either downloaded from the market place or created previously, import and apply the template. The mesh in the right scene will automatically deform to wrap the model in the middle scene.

Deform By Point/Face Matching​

Using the operations 1 2 3 4, deform the source mesh to wrap the target model completely.

6. Transfer Textures​

Transfer the textures in the middle scene to the mesh or some faces in the right scene.

7. Export​

Export the source mesh or its some faces in the right scene to a fbx or obj file.
