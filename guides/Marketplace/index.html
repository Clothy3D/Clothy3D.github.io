Marketplace Guide

In the marketplace, you can

  • discover, get and buy, share and sell interchangeable cloth and base character, template and tool assets;
  • review any published assets, and report any issues with them;
  • publish requests, make and accept offers;
  • send/receive messages to/from other users.

Requirements for Sharing and Selling:

  • you are required to have an account on at least one sharing platform such as Google drive

Discover Assets​

You can search for assets.

Get And Buy​

  1. Buy any assets including free assets with price of 0.

    You must buy an asset even it is free, otherwise the Clothy3D tools will not be able to open it.

No Real Trading In Beta Testing

In the current beta testing, you can buy an non-free asset using a testing credit card number, and do not use your real credit card. Real asset trading will be supported in future.

  1. Download the assets using their download links in the listing description.

Review Published Assets​

You can review assets by comments and rating.

Report Any Issues With A Published Asset​

Your report will be sent to the Clothy3D administrator.

Share Or Sell By Publishing An Asset​

Step 1. Generate An Asset With An ID of 0 and Its Images​

  1. Export an asset with an ID of 0 using any Clothy3D tools
  2. Create or get 1-3 images demonstrating the asset. The images are recommended to have size of 512x512. You can use the buttons shown in the following figure to create a 512x512 image capturing a viewport.

Step 2. List The Asset​

  1. List the asset:

  • Select category.

    You must select the category and can not select it to the "Tools" category.

  • Select the images

  • Write title

  • Set a price.

    If you set the price to 0, then the price will be changed to the allowed maximal price in order for you to be able to edit or delete the listing later.

  • Select license

  • Write description

  1. Submit the listing

    The Clothy3D administor will send you a message informing you the ID issued to asset.

  2. Check your received messages, find the assigned ID of the submitted asset.

Step 3. Change Asset ID​

Use the Change Asset ID tool to change the asset ID from 0 to the issued value.

Step 4. Upload The Asset and Get Download Links​

  1. Upload the asset file (.zip or .dfm file containing the issued ID) to one or more file sharing platforms such as Google Drive
  2. Generate the download links of the uploaded file.
  3. (optionally and recommended) Convert the download links to a direct download links.

Step 5. Edit And Publish (Unhide) Listing Or Delete It​

  1. Edit the listing and save the Changes

  • Modify description

    Add the download links in the description, and do any modifications you need.

  • Change price if you need

    You can change the price to 0 if you want to share it as a free aset. But if you do so, then you can not edit the listing again or delete it. So you must ensure all the listing content is correct before doing so.

    The mininal acceptable zon-zero price is $1 (1 US Dollar).

No Real Trading In Beta Testing

In the current beta testing, you can set the price to non-zero， but will not be paid by real money. Real asset trading will be supported in future.

  1. Publish (Unhide) the listing

    You can also delete the listing here if you need.

Publish Requests, Make And Accept Offers​

Any user can publish requests and accept offers.

Any user becomes a vendor after publishing a free asset, and then can make offers to any requests.

All requests will expire after 30 days and will be deleted afterwards.

Tips​

Generate Direct Download Links​

You can search "how to get direct download link" on www.google.com to find instructions.

Replace Shared Files Without Changing Download Links​

You can search "how to replace shared file without changing download link" on www.google.com to find instructions.

Send/Receive Messages To/From Other Users​

All messages will be kept for 7 days and will be deleted afterwards.
